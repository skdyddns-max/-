📚 내 프로젝트

목표 기술·아동별로 프로젝트를 여러 개 만들어 관리할 수 있습니다. 모든 프로젝트는 이 브라우저에 저장됩니다.

❓ 처음이신가요? — 5단계 사용법 보기
1. ⚙ 설정에서 Google AI Studio 키를 넣고 "대본 생성 방식"을 Gemini로 선택하세요. (대본은 무료 키로도 가능, Veo 영상은 유료 등급 필요. 키가 없으면 "프롬프트 복사 모드"로도 쓸 수 있어요)
2. 아래에서 목표 화용 기술과 아동 정보를 입력하고 시나리오 생성을 누르세요.
3. ② 대본에서 내용을 확인·수정하고, ③에서 장면별 영상(Veo)과 더빙(TTS)을 만들어 저장하세요.
4. ⑤에서 치료 플레이어를 내려받아 태블릿에서 열고, 영상을 지정해 세션을 진행하세요. 퀴즈 결과가 자동 기록됩니다.
5. 프로젝트는 📚 목록에서 아동·목표별로 여러 개 관리할 수 있어요. 인쇄용 활동지도 ⑤에서 내려받을 수 있습니다.

① 치료 목표 설정

목표 화용 기술과 아동 정보를 입력하면, 비디오 모델링용 시나리오 전체(대본·영상 프롬프트·음성 대본·토론 질문·퀴즈)가 한 번에 생성됩니다.

⑤ 내보내기

🎯 치료용 인터랙티브 플레이어

영상 재생 → 장면 사이 퀴즈 → 세션 요약 → 토론 질문까지 진행되는 독립 실행형 HTML 파일입니다. 태블릿이나 PC에서 파일을 열고, 만들어 둔 영상 파일(+선택: 더빙 음성)을 장면별로 지정하면 바로 치료 세션에 사용할 수 있습니다. 더빙 음성을 지정하면 영상과 동시에 재생됩니다. (인터넷 연결 불필요)

📊 진행 기록: 아동 이름을 입력하고 세션을 마치면 퀴즈 시도 횟수·첫 시도 정답률이 자동 저장되고, 지난 기록을 CSV(엑셀)로 내려받을 수 있습니다. 🔁 장면 다시 보기 버튼으로 반복 시청도 지원합니다.

🎤 말하기 연습: 퀴즈가 끝나면 아동이 주인공의 대사를 직접 말하고 녹음해 들어보는 역할극 단계가 이어집니다(플레이어에서 켜고 끌 수 있음). "스스로/도움 받아 말했어요" 평가가 기록에 함께 남습니다.

🖨 인쇄용 활동지

표지(목표·진행 순서) → 장면별 대본·토론 질문(기록란) → 퀴즈 정답지(치료사용) → 수기 세션 기록지로 구성된 A4 인쇄용 문서입니다. 내려받은 파일을 열고 "인쇄 / PDF 저장" 버튼을 누르세요.

💾 프로젝트 백업

프로젝트 데이터(JSON)를 저장해 두면 다른 기기에서 불러와 이어서 작업할 수 있습니다.

🃏 감정 카드 만들기

표정 읽기·감정 어휘 훈련용 카드입니다. 이야기 속 주인공이 여러 감정의 표정을 짓는 그림을 생성해, 화면으로 보여주거나 인쇄해서 사용하세요.

💡 활용법: 카드를 보여주고 "어떤 기분일까?" 묻기 → 이야기 속 상황과 연결하기("도도가 언제 이런 표정이었지?") → 거울 보며 표정 따라 하기.

이미지 생성은 ⚙ 설정의 Gemini API 키(유료 등급)를 사용하며 장당 소액 과금됩니다. 카드는 저장 버튼을 누르기 전까지는 페이지를 닫으면 사라집니다.

🫧 마음 읽기 (속마음)

마음이론·추론 영역 활동입니다. 등장인물이 겉으로 한 말·행동과 진짜 속마음이 다른 상황을 보여주고, 아동이 표정·상황 단서로 숨은 마음을 추론하게 합니다.

💡 활용법: 상황과 겉말을 보여주고 "진짜 마음은 뭘까?" 묻기 → 표정·단서 근거 찾기 → 🫧 속마음 뒤집어 확인 → "너도 이런 적 있었어?" 연결.

🧭 어떻게 할까? (상황 선택)

사회적 문제 해결 영역 활동입니다. 곤란한 상황을 주고 여러 대처 방법 중 하나를 고르면, 그 선택이 어떤 결과로 이어지는지 보여줍니다.

💡 활용법: 상황을 읽어주고 "너라면 어떻게 할래?" 묻기 → 선택 → 결과 확인 → "다른 방법은 어떤 결과일까?" 비교 → 가장 좋은 방법 함께 찾기.
