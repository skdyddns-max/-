# Brand OS 볼트 — 나용운의 세계관

이 폴더는 **옵시디언(Obsidian) 볼트**로 열도록 설계된 세계관 문서 시스템입니다.
2026-07-16의 Brand OS v0.1 초안을 원자 노트로 분해하고, 초인지(메타) 수준에서 분석한 뒤, v0.2로 재조합한 결과물입니다.

## 여는 법

1. Obsidian 실행 → "Open folder as vault" → 이 `brand-os` 폴더 선택
2. **처음이면 [[01_쉽게 읽기]]부터** — 용어 번역표와 5분 요약이 있습니다
3. 그다음 [[00_HOME]] 노트가 전체 지도 (Ctrl/Cmd+O → "00_HOME")
4. 그래프 뷰(Ctrl/Cmd+G)를 열면 세계관의 연결 구조가 보입니다

## 구조 — 사고의 4단계

| 폴더 | 단계 | 하는 일 |
|---|---|---|
| `10_원문` | 보존 | v0.1 원문을 그대로 보관 (수정 금지, 스냅샷) |
| `20_원자노트` | 분해 | 원문을 재사용 가능한 최소 단위로 쪼갬 |
| `30_메타분석` | 초인지 | 문서 *내용*이 아니라 문서를 쓴 *사고방식*을 분석 |
| `40_재조합` | 종합 | 분석 결과로 v0.2를 다시 조립 |
| `50_템플릿` | 실행 | 킥오프·효과 기록·분기 점검 — 복사해서 쓰는 장치 |

## 태그 체계

- `#코어` — 미션·비전·정체성 (거의 안 변하는 것)
- `#원칙` — 판단 기준 (느리게 변하는 것)
- `#제품` — 만드는 것 (빠르게 변하는 것)
- `#메타` — 초인지 분석 노트
- `#재조합` — 종합·제안 노트
- `#템플릿` — 복사해서 쓰는 실행 장치
- `#열린질문` — 아직 답하지 않은 질문

## 개인정보 원칙

이 볼트는 공개 저장소에 있으므로 아동 실명을 쓰지 않습니다.
원문의 실명 두 건은 **H 아동 / J 아동**으로 익명화했습니다. → [[메타 4 — 빈칸과 질문]]

## 갱신 규칙

- 원자 노트는 자유롭게 수정 (살아있는 문서)
- `10_원문`은 절대 수정하지 않음 — 버전이 바뀌면 새 스냅샷 추가 (현재 확정본: v0.2, 다음은 분기 점검 후 v0.3)
- 새 제품·새 생각이 생기면: 원자 노트 추가 → [[00_HOME]]에 링크 → 분기마다 메타분석 노트 재검토
