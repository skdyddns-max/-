# 현재 프로젝트 — 지금 만들고 있는 것

## 1. AAC 프로젝트 (N=1 단계)

- **H 아동 맞춤 AAC**
- **J 아동 맞춤 AAC**

[[원칙 3 — 한 아이를 위해 만들고 많은 아이에게 확장한다]]의 1단계를 실명 그대로 실행 중인 프로젝트. 두 사례가 쌓이면 공통 패턴 추출 → [[미래 프로젝트]]의 "AAC 자동 생성기"로 건너갈 재료가 된다.

> 📌 두 아이의 AAC에서 **무엇이 같고 무엇이 달랐는지** 기록해 둘 것 — 그 차이 목록이 곧 자동 생성기의 설정 화면이 된다.

## 2. AI 워크시트 분석 브리핑 생성기 (N=1 → N=다수 전환 후보)

자동 생성 항목: 정반응률 · 오반응 패턴 · 강점/난점 · 임상적 해석 · 보호자용 리포트

이 도구는 이중 역할을 가진다:

1. **제품** — 치료사의 문서 작업 시간을 줄인다
2. **검증 인프라** — [[원칙 4 — 기술보다 치료 효과를 먼저 생각한다]]가 요구하는 "효과 측정"의 데이터 소스. 다른 모든 제품의 효과를 이 도구가 잰다.

## 3. (레포에 이미 존재하는 것) 화용 스튜디오

이 저장소의 `studio.html` — AI 영상 화용언어치료 제작 도구. 대본 생성 → Veo 영상 → 퀴즈 → 치료 플레이어 → 진행 기록(정답률 CSV)까지. v0.1 문서에는 빠져 있으나 실제로는 세계관에 가장 부합하는 완성물이다 → [[메타 3 — 긴장과 모순]]의 긴장 2에서 다룸. **v0.2 문서에 편입할 것.**
