# 마음터칭 · 마음지수 진단

**비본어게인 연담(然談)** — 자연스럽고 깊은 마음

성인 대상 심리 케어 서비스 **마음터칭**의 모바일 설문 페이지입니다.
자연 속 전원주택에서 진행되는 케어 프로그램을, 간단한 설문으로 맞춤 추천해 드립니다.

## 접속

- 웹: https://skdyddns-max.github.io/-/ (GitHub Pages 자동 배포)
- 모바일 최적화 페이지이며, 브로셔 QR 코드로 연결됩니다.

## 무엇을 하나요?

1. **마음지수 진단** — 마음 날씨 + 우울감 셀프 체크(약 3분) → 0~100점
2. **자기 인식 체크리스트** — 내가 보는 나 vs 남이 보는 나 비교
3. **맞춤 프로그램 추천** — 싱잉볼 명상 · 아로마테라피 스트레칭 · 사운드 테라피 · 1:1 맞춤 세션
4. **예약** — 결과 화면 캡처 후 네이버폼으로 신청

프로그램은 강화의 자연 속 전원주택에서 진행되며, 대중교통 이용 시 **강화터미널 픽업**이 가능합니다.

## 화용 스튜디오 (AI 영상 화용언어치료 제작 도구)

`studio.html` — 화용언어치료용 비디오 모델링 콘텐츠를 만드는 통합 제작 도구입니다.

- 웹: https://skdyddns-max.github.io/-/studio.html
- **📚 프로젝트 목록** — 목표 기술·아동별로 프로젝트 여러 개 생성·전환·복제·삭제
- **① 목표 설정** — 목표 화용 기술을 **7대 화용 영역**(의사소통 기능 · 대화 기술 · 사회적 상호작용·예절 · 비언어적 의사소통 · 마음이론·추론 · 감정·자기조절 · 사회적 문제 해결)으로 묶어 세부 기술 40여 종 제공. 선택한 영역별 지도 지침이 대본 생성에 자동 반영됨. 연령·상황·아동 특성도 입력
- **② 대본** — AI가 시나리오(대사·비언어 단서·토론 질문) 생성, 장면별 수정 요청 가능. 생성 엔진은 Claude API / **Gemini API(무료 등급 가능)** / 프롬프트 복사 모드 중 선택
- **③ 영상 제작** — Gemini API 키 하나로: 장면별 **Veo 영상 생성(한국어 대사 오디오 포함 옵션) → 미리보기 → mp4 저장** + **Gemini TTS 더빙 생성 → 미리듣기 → wav 저장**. 수동 경로(프롬프트 복사 → Kling·Sora 등)도 그대로 지원
- **④ 퀴즈** — 장면 사이에 나올 선택형 질문 편집
- **🃏 감정 카드** — 이야기 주인공(또는 기본 캐릭터)이 8가지 감정 표정을 짓는 카드를 Gemini 이미지 생성으로 제작 → PNG 저장 · A4 인쇄용 카드 시트(절취선 포함) · 플레이어에 내장되어 **감정 놀이 모드**(카드 보고 감정 단어 고르기, 결과 기록 포함)로 사용
- **🫧 마음 읽기(속마음)** — 마음이론·추론 영역 전용 활동. 등장인물이 겉으로 한 말·행동과 **진짜 속마음**이 다른 상황 카드를 AI로 생성 → 뒤집어 속마음 확인 · A4 인쇄용 카드 시트 · 플레이어에 **마음 읽기 게임 모드**(표정·상황 단서로 숨은 마음 3지선다 추론, 결과 기록 포함)로 내장
- **🧭 어떻게 할까?(상황 선택)** — 사회적 문제 해결 영역 전용 활동. 곤란한 상황과 3가지 대처 방법(좋은/애매한/안 좋은)·각 결과를 AI로 생성 → A4 인쇄용 카드 시트 · 플레이어에 **어떻게 할까? 게임 모드**(방법 선택 → 결과 확인, 여러 선택을 비교하며 가장 좋은 방법 찾기, 결과 기록 포함)로 내장
- **⑤ 내보내기** — 영상 재생 → 퀴즈 → 세션 요약 → 토론 질문으로 진행되는 독립 실행형 치료 플레이어 HTML 다운로드
  - 플레이어에 아동별 진행 기록 내장: 퀴즈 시도 횟수·첫 시도 정답률 자동 저장, 지난 기록 목록, CSV(엑셀) 내려받기, 장면 다시 보기
  - 장면별 더빙 음성(wav/mp3)을 지정하면 영상과 동시 재생
  - **🎤 말하기 연습(역할극)**: 퀴즈 후 아동이 대사를 직접 말하고 녹음·재생, 치료사 평가(스스로/도움)가 기록에 저장
  - 홈 화면 설치 지원(PWA): 태블릿·폰에서 앱처럼 실행
  - **인쇄용 활동지**: 표지·장면별 대본·토론 질문(기록란)·퀴즈 정답지·수기 세션 기록지 (A4 인쇄/PDF)

생성 방식은 두 가지: Claude API 키를 넣으면 자동 생성, 키가 없으면 프롬프트를 복사해 claude.ai에 붙여넣고 결과 JSON을 되붙이는 방식으로 동작합니다. 모든 데이터는 브라우저(localStorage)에만 저장됩니다.

## 톤메이커 (AI 원탭 사진 보정)

`tonemaker.html` — 사진을 올리면 AI가 톤을 잡아주고, 손과 말로 다듬고, 여러 장에 똑같이 입히는 원탭 보정 도구입니다.

- 웹: https://skdyddns-max.github.io/-/tonemaker.html
- **✨ 원탭 자동 톤** — 사진을 올리면 즉시 톤 후보 4개(자연스럽게·화사하게·필름 감성·쨍하게)를 실제 미리보기 썸네일로 제시. **AI 톤 추천**을 누르면 Claude가 사진을 직접 보고 장면 진단 + 사진에 맞춘 정밀 후보를 생성
- **🎛 직관 다듬기** — 숫자 없이 "밝게↔어둡게" 식 손잡이 6개로 조정. 전문가 모드를 펼치면 노출·대비·밝은/어두운 영역·색온도·HSL·페이드·그레인·할레이션 등 세부 값과 히스토그램(클리핑 경고 포함) 표시
- **🧑 인물·피부 보정** — 피부색 영역을 자동 감지해 그 부분만 적용: 피부 결 매끄럽게 · 모공/잡티 완화(어두운 점만 들어올림) · 피부 화사하게. 눈·머리카락·배경 디테일은 유지. AI 톤 추천·대화 보정도 인물 사진에서 자동 활용
- **💬 자연어 대화 보정** — "하늘을 더 파랗게", "인물 얼굴만 밝게" 말하면 Claude가 사진과 현재 톤을 보고 값을 다듬고 이유를 설명. 부분 보정(마스크)까지 자동 생성
- **🖼 레퍼런스 톤 매칭** — 따라하고 싶은 감성 사진을 올리면 그 톤을 내 사진에 이식. ⚡빠른 매칭(로컬 통계: 밝기·대비·채도·색온도)과 🤖AI 정밀 매칭(색 물들임·필름 질감까지) 두 경로
- **✂ AI 구도 제안** — Claude가 3분할·수평 보정·주제 강조 관점으로 크롭 1~3개를 이유와 함께 제안, 탭 한 번 적용·원본 복귀 (비파괴, 회전 수평 보정 포함)
- **🎯 부분 마스킹 보정** — ◯지점(사진 탭해서 배치·이동)·▽/△그라데이션·☀밝은 곳·🌙어두운 곳 마스크(최대 3개)로 일부만 밝기·대비·채도·온도 조정. "영역 바깥 적용"으로 배경만 어둡게도 가능
- **🪄 생성형 리터치** — AI가 픽셀을 직접 다시 그림: 잡티 클린·배경 흐림·방해물 제거·화질 개선 + "전선 지워줘" 직접 지시. 비파괴 스택(한 단계씩 되돌리기), 리터치 결과 위에 톤 보정이 그대로 적용. 엔진: Gemini 이미지 편집(`gemini-2.5-flash-image`, 무료 등급 가능)
- **🎨 무드 카드** — 시네마틱·새벽 감성·여름 쨍함·빈티지 필름·가을 노을·흑백 (스플릿 토닝 컬러 그레이딩 내장), 강도 0~100% 조절
- **🎬 .cube LUT** — 라이트룸·다빈치·프리미어용 3D LUT를 불러와 실시간 적용(강도 조절, WebGL 트라이리니어 보간, 65³은 자동 리샘플). 반대로 **현재 톤을 .cube로 내보내** 영상 편집 프로그램에서 그대로 사용 가능. LUT 포함 프리셋 저장·파일 공유 지원
- **⭐ 프리셋 + 📦 일괄 보정** — 마음에 든 톤을 저장하고 올린 사진 전체에 한 번에 적용. 사진마다 원본 밝기 편차를 자동 보정하는 옵션 포함
- **◧ 비교** — 원본/보정 드래그 슬라이더, 원본 꾹 눌러 보기, 되돌리기 무제한 (원본 무손실)
- 보정 연산은 전부 브라우저 WebGL에서 처리(무료·즉시). AI 분석 시에만 축소본이 전송되며, 저장 시 EXIF(위치정보 등) 자동 제거
- 생성 방식: Claude API 키 자동 모드 / 키 없이 프롬프트 복사 모드

## 디테일 — AI 상세페이지 스튜디오

`product.html` — 제품 사진 한 장 또는 한 줄 설명만으로 **구매를 부르는 SNS·커머스 상세페이지**를 통째로 만들어 주는 서비스입니다. 제품을 소개하는 **랜딩페이지**와 실제 생성 **도구**가 한 파일에 들어 있어, 지속적인 리뉴얼로 개선을 이어갈 수 있는 구조입니다.

- 웹: https://skdyddns-max.github.io/-/product.html
- **랜딩페이지** — 서비스 소개(히어로·작동 방식 3단계·기능·예시·CTA). "만들기 시작"으로 생성 도구에 진입 (`#studio`)
- **도구 · 입력** — 제품명·카테고리·한 줄 설명·타깃·가격·톤&무드·판매 채널 + 제품 사진(선택, 최대 3장). **제품명을 적으면 카테고리 자동 감지**(책·출판물/교육·자료/뷰티/식품/패션/리빙/디지털/반려·키즈/서비스 등), 카테고리별 **세부 항목**(책=저자·판형·제본·목차·독자, 교육자료=목표·연령·구성·이용범위, 화장품=성분·피부타입·용량 …)이 열려 입력한 사실이 스펙·구성·혜택에 그대로 반영됨
- **이미지** — 올린 사진(책 표지·자료 등)을 결과에 **즉시 반영**(AI 연출 성공 시 감성 컷으로 대체, 실패해도 원본 유지). 이미지 모델은 계정이 실제 가진 것으로 자동 조회, 이미지 내 글자는 렌더링 금지(오타 방지)
- **카피 생성** — 소비자가 결제 직전 유심히 보는 **다섯 가지 불안**을 정면으로 해소하도록 설계: ① 실물 신뢰(성분·소재·크기) ② 적합성(추천 + **정직한 비추천**) ③ 가치 정당화(구성·포함) ④ 사회적 증거(후기) ⑤ 리스크 제거(배송·교환·환불·보증). 후킹 헤드라인·공감·혜택·**성분/원재료·크기 안내·스펙·적합성·구성·후기·안심·FAQ·CTA**까지 한 번에. 판매량·별점 같은 수치는 지어내지 않고 '교체용'으로 정직하게 표기. 엔진은 **Gemini API(`gemini-2.5-flash`, 무료 등급 가능)**
- **이미지 생성** — 올린 제품 사진을 **참조**로 그 제품의 실제 모습을 유지한 채 히어로·라이프스타일·디테일 감성 컷을 생성(`gemini-2.5-flash-image`, Nano Banana). 사진이 없으면 설명만으로 생성. 톤&무드에 맞춰 조명·색감 자동 반영
- **디자인 테마 6종** — 결과물이 토큰 기반 디자인 시스템 위에서 동작: 지면·잉크·헤어라인·라운드·헤드라인 서체(명조/고딕)·자간·히어로 구도까지 테마가 통째로 바뀜. **에디토리얼 / 감성 세리프(블러시+명조) / 클린 프로(쿨 그레이+블루) / 팝 볼드(화이트+탠저린+900) / 럭셔리 다크(올-블랙+골드+명조) / 내추럴(크림+그린)**. 톤&무드를 고르면 자동 매칭, **AI가 제품을 읽고 어울리는 테마를 추천**(AI 배지), 미리보기에서 원클릭으로 갈아입혀 비교. **이미지 무드도 테마를 따라감** — 테마 변경 후 재생성하면 사진 조명·색감이 페이지와 맞춰짐
- **프로 기획·전환 장치** — 실무 용어 체계로 생성: 훅(페인포인트) → 헤드카피/서브카피 → **USP 뱃지 3개** → 베네핏 → 소셜프루프 → CTA. **헤드카피 A/B 2안** 생성 후 미리보기에서 원클릭 전환(A/B 테스트), 스크롤 중간 이탈을 막는 **중간 CTA**, 항상 손 닿는 **스티키 구매 바**(가격+CTA, 편집 시 자동 동기화)
- **퍼블리싱 품질** — below-the-fold 이미지 **레이지 로딩**, 전 이미지 **alt 텍스트** 자동, 업로드 이미지 **webp 인코딩**(미지원 브라우저는 jpeg 폴백), 반응형·모바일 최적화
- **미리보기·편집** — 디바이스 프레임 안에 실시간 렌더. 문구는 클릭해 바로 수정, 이미지는 마우스를 올려 **재생성·업로드·PNG 저장**
- **📱 SNS 세트** — 캔버스 합성으로 **카피가 얹힌 완성 컷** 생성: 피드 4장(1080×1350 — 커버·훅·USP·CTA) + 스토리 1장(1080×1920). 테마 색·서체가 그대로 적용되고, **인스타 캡션(첫 줄 훅) + 해시태그 15~20개**까지 자동 생성·원클릭 복사 — 그대로 업로드하면 됨
- **내보내기** — 이미지가 내장된 **독립 실행형 HTML** 다운로드(어느 호스팅에나 업로드 가능) + **SEO 완비**: meta description, Open Graph, JSON-LD Product 구조화 데이터(가격 포함), 이미지 일괄 저장

디자인은 웜 차콜 + 페이퍼 + 마리골드 액센트의 "에디토리얼 아틀리에" 톤(단일 파일, 빌드 없음).

**생성 엔진은 세 가지 중 선택** (⚙ 설정에서 전환):
- **Gemini** (기본) — `gemini-2.5-flash` + `gemini-2.5-flash-image`. Google AI Studio에서 무료 발급 가능
- **GPT · OpenAI** — `gpt-4.1-mini` + `gpt-image-1`(참조 사진은 images/edits로 실물 유지). platform.openai.com에서 발급하며 **ChatGPT 구독과 별개로 API 크레딧 필요**, gpt-image-1은 계정에 따라 조직 인증 필요
- **프롬프트 복사 — 키 없음** — 페이지가 만든 프롬프트를 쓰고 있는 **ChatGPT(구독)·Claude·Gemini 웹**에 붙여넣고 답변(JSON)을 되붙이면 완성. 이미지는 각 자리에서 '재생성'으로 그 컷용 프롬프트를 복사해 챗봇으로 만들고, 자리를 클릭해 직접 업로드

이미지 자리는 어느 엔진에서든 **직접 업로드**(올리기)를 지원 — 실제 제품 사진을 그대로 쓸 수도 있습니다.

모든 처리는 브라우저 안에서만 이뤄집니다(백엔드 없음). 키는 이 브라우저(localStorage)에만 저장됩니다.

## 워크시트 채점 브리핑 (AI 워크시트 분석 & 결과 브리핑 생성기)

`worksheet.html` — 아이가 푼 학습지 사진을 올리면 AI가 채점·분석하고, 치료사용·보호자용 결과 브리핑을 자동으로 만들어 주는 도구입니다.

- 웹: https://skdyddns-max.github.io/-/worksheet.html
- **입력** — 워크시트 사진·스캔본(최대 5장) + 정답 기준 3가지 방식: ✍️ 텍스트 입력 / 🖼 정답지 사진 첨부 / 🤖 AI가 문제를 읽고 직접 판독. 세션 날짜·영역(조음·수용언어·읽기·수학 등)·과제명·회기 목표·수행 시 특이사항(촉구 제공 등) 기록
- **AI 분석** — 문항별 채점(정반응·오반응·무반응·**판독불가** 구분 + **자기수정** 추적, 글씨가 흐리면 추측하지 않음) → **정반응률**(판독불가 제외 재계산)과 **수행 수준 판정**(독립 수행≥90 / 습득 진행 70~89 / 연습 40~69 / 도입 <40). **오반응 패턴**은 유형·빈도만이 아니라 **발생 조건·일관성(일관적/비일관적)·기제 가설·다음 회기 검증 프로브**까지, **강점/난점은 근거 문항·수치 인용 필수**. **임상적 해석은 4단 구조**(수행 수준 판단 → 오류 기제 가설(경쟁 가설 병기, 각각 근거·확인 방법) → 수행 흐름(전/후반부 정확도 비교로 주의·피로 관찰) → 종합 소견), 권고는 **측정 가능한 조건–행동–기준 목표 + 활동 + 근거** 표로 출력. 판독 신뢰도가 낮으면 경고 표시
- **촉구 수준 기록** — 문항마다 촉구 5단계(독립/구어/몸짓/시범/신체) 기록. AI는 치료사가 특이사항에 적은 촉구만 반영(사진으로 추측 금지)하고, 분석 후 문항표에서 탭으로 수정. **독립 정반응률**을 별도 산출해 촉구 포함 수치와의 차이(촉구 의존도)를 하이라이트·치료사 브리핑에 표기, 보호자용은 "혼자 힘으로/선생님과 함께 해낸 문제"로 구분
- **문항표 수동 교정** — 결과 배지(정↔오↔무반응↔판독불가)와 촉구 배지를 탭하면 정반응률·독립 정반응률·브리핑 수치가 즉시 재계산
- **결과 브리핑 (고정 템플릿 2종)** — 🩺 **치료사용 상세**(수행 요약·문항표·오류 패턴·강점/난점·임상적 해석·권고) / 👨‍👩‍👧 **보호자용**(전문용어 없이: 오늘 한 활동·잘한 점·함께 연습하면 좋은 점·집에서 하는 5~10분 활동·응원 한마디·별점 성취 표시). 본문은 클릭해 자유롭게 수정(자동 저장, 원문 복귀 가능)
- **인쇄 / PDF 저장** — 브리핑을 A4 인쇄 서식으로 바로 출력(브라우저 인쇄 대화상자에서 PDF 저장). 텍스트 복사로 알림장·메신저 전달도 가능
- **아동별 기록** — 아동 여러 명 관리, 세션 자동 저장(썸네일 포함), 열람·삭제, **CSV(엑셀) 내려받기**
- **변화 추이 시각화** — 아동·영역 필터로 정반응률 라인 차트(**습득 기준선 80%** + 촉구 기록이 있으면 **독립 정반응률 점선** 병행 표시), 오류 유형 누적 분포 차트, 자동 분석 요약: 첫↔최근 변화량, **최근 3회 평균과 기준 대비**(3회 연속 충족 시 유지·일반화 전환 제안), 연속 상승/하락 감지, **오류 유형 변화 추적**(사라진/지속/새로 나타난 오류) (SVG 자체 렌더, 라이브러리 없음)
- 생성 방식: Claude API 키 자동 모드 / 키 없이 프롬프트 복사 모드(claude.ai에 사진 첨부 → JSON 되붙이기). 사진은 분석 시에만 축소 전송되고, 모든 기록은 브라우저(localStorage)에만 저장. 본 분석은 참고자료이며 치료사의 임상적 판단을 대체하지 않습니다
