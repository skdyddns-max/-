# 마음터칭 · 마음지수 진단

**비본어게인 연담(然談)** — 자연스럽고 깊은 마음

성인 대상 심리 케어 서비스 **마음터칭**의 모바일 설문 페이지입니다.
자연 속 전원주택에서 진행되는 케어 프로그램을, 간단한 설문으로 맞춤 추천해 드립니다.

## 접속

- 웹: https://skdyddns-max.github.io/-/ (GitHub Pages 자동 배포)
- 모바일 최적화 페이지이며, 브로셔 QR 코드로 연결됩니다.

## 무엇을 하나요?

1. **마음지수 진단** — 마음 날씨 + 우울감 셀프 체크(약 3분) → 0~100점
2. **자기 인식 체크리스트** — 내가 보는 나 vs 남이 보는 나 비교
3. **맞춤 프로그램 추천** — 싱잉볼 명상 · 아로마테라피 스트레칭 · 사운드 테라피 · 1:1 맞춤 세션
4. **예약** — 결과 화면 캡처 후 네이버폼으로 신청

## 오시는 길

프로그램은 강화의 자연 속 전원주택에서 진행되며, 대중교통 이용 시 **강화터미널 픽업**이 가능합니다.
(추후 제작할 브로셔에도 픽업 안내를 포함할 것)

## 화용 스튜디오 (AI 영상 화용언어치료 제작 도구)

`studio.html` — 화용언어치료용 비디오 모델링 콘텐츠를 만드는 통합 제작 도구입니다.

- 웹: https://skdyddns-max.github.io/-/studio.html
- **📚 프로젝트 목록** — 목표 기술·아동별로 프로젝트 여러 개 생성·전환·복제·삭제
- **① 목표 설정** — 목표 화용 기술·연령·상황 입력
- **② 대본** — AI가 시나리오(대사·비언어 단서·토론 질문) 생성, 장면별 수정 요청 가능. 생성 엔진은 Claude API / **Gemini API(무료 등급 가능)** / 프롬프트 복사 모드 중 선택
- **③ 영상 제작** — Gemini API 키 하나로: 장면별 **Veo 영상 생성(한국어 대사 오디오 포함 옵션) → 미리보기 → mp4 저장** + **Gemini TTS 더빙 생성 → 미리듣기 → wav 저장**. 수동 경로(프롬프트 복사 → Kling·Sora 등)도 그대로 지원
- **④ 퀴즈** — 장면 사이에 나올 선택형 질문 편집
- **⑤ 내보내기** — 영상 재생 → 퀴즈 → 세션 요약 → 토론 질문으로 진행되는 독립 실행형 치료 플레이어 HTML 다운로드
  - 플레이어에 아동별 진행 기록 내장: 퀴즈 시도 횟수·첫 시도 정답률 자동 저장, 지난 기록 목록, CSV(엑셀) 내려받기, 장면 다시 보기
  - 장면별 더빙 음성(wav/mp3)을 지정하면 영상과 동시 재생
  - **인쇄용 활동지**: 표지·장면별 대본·토론 질문(기록란)·퀴즈 정답지·수기 세션 기록지 (A4 인쇄/PDF)

생성 방식은 두 가지: Claude API 키를 넣으면 자동 생성, 키가 없으면 프롬프트를 복사해 claude.ai에 붙여넣고 결과 JSON을 되붙이는 방식으로 동작합니다. 모든 데이터는 브라우저(localStorage)에만 저장됩니다.

## 구조

- `index.html` — 마음지수 진단 페이지 (별도 빌드·백엔드 없음)
- `studio.html` — 화용 스튜디오 (단일 파일, 백엔드 없음)
- `.github/workflows/deploy-pages.yml` — main 푸시 시 GitHub Pages 자동 배포

## 설정

예약용 네이버폼 링크는 `index.html` 상단의 `NAVER_FORM_URL` 값에 넣으면 됩니다.
