상세페이지 스튜디오

제품 한 장·한 줄이면 → 구매를 부르는 SNS·커머스 상세페이지

1 제품 정보
아는 만큼만 채워도 AI가 나머지를 마케팅 관점으로 완성합니다.
2 제품 사진 (선택)
사진을 올리면 그 제품 그대로를 살려 감성 컷으로 재탄생시켜요. 없으면 설명만으로 생성합니다.
📷 사진 올리기
클릭 또는 드래그 · 최대 3장
📄 미리보기
🛍️

여기에 상세페이지가 나타납니다

왼쪽에 제품 정보를 넣고 상세페이지 생성을 누르면
구매를 부르는 카피 + 감성 이미지가 자동으로 완성돼요.
완성된 문구는 클릭해서 바로 수정할 수 있습니다.

⚙️ Gemini API 키

키는 이 브라우저에만 저장되고 서버로 전송되지 않습니다.

Google AI Studio에서 무료로 발급할 수 있어요. 텍스트는 gemini-2.5-flash, 이미지는 gemini-2.5-flash-image 모델을 사용합니다.

❓ 사용법

1. ⚙️ API 키에서 Gemini 키를 넣어요 (무료 등급 가능).
2. 제품명 + 한 줄 설명을 넣고, 원하면 제품 사진을 올려요.
3. 상세페이지 생성 → 카피와 감성 이미지가 자동 완성.
4. 문구는 클릭해서 바로 수정, 이미지는 마우스를 올려 재생성·저장.
5. ⬇️ HTML 저장으로 이미지가 박힌 독립 실행형 상세페이지를 내려받아 어디든 올릴 수 있어요.

🔒 모든 입력·결과는 브라우저 안에서만 처리됩니다. 별도 서버·백엔드 없음.
